### Problem Description

To Develop an environment contain some dirt and the agent is going to clean. The aim is to clean the dirty place.

### Action Space

### Sample Action

### Reward Function

- +1 - when an agent move to the right side and the dirt using vaccum
- 0 - Otherwise
